Declared in C4GL1Renderer.h

Overview

This document describes the C4GL1Renderer class which should be used / subclassed to create your own rendering algorithms for a C4GL object.

Instead of manipulating this object’s initialization methods, you can instantiate variables using the setup method. The setup method gets called at the end of the init method executed by the superclass C4EAGLES1Renderer.
